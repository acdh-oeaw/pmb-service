Export

Die in der PMB verzeichneten Inhalte stehen frei zur Verfügung. Sie sind mit der Lizenz CC BY 4.0 versehen und können ohne Rücksprache verwendet und weiter bearbeitet werden.

Einzelne Einträge können über die API (die JSON- oder TEI-Symbole bei einem Eintrag) geladen werden.

Die REST-API ist über "https://pmb.acdh.oeaw.ac.at/apis/api/" anzusprechen, beispielsweise https://pmb.acdh.oeaw.ac.at/apis/api/entities/place/5/".

Als TEI-XML-Listen können sie zur Gänze geladen werden (rechte Maustaste: »speichern unter«):

  • Personen.
  • Werke.
  • Orte.
  • Institutionen.
  • Ereignisse.

Wir freuen uns, wenn sich weitere Unternehmungen anschließen und die PMB für die Erfassung ihrer Entitäten verwenden würden und bitten um Kontaktaufnahme.

Im Ordner media finden sich noch weitere tagesaktuelle Auswertungen, die beispielsweise für die Visualisierung hilfreich sein können. Eine einfache Möglichkeit, Inhalte der PMB zu visualisieren, kann durch den Service APIS-Hub erfolgen. (Beispielsweise: "Source entity type" auf "Person" stellen, "Target entity type" auf "event" und auf "Load" drücken.) Eine andere Möglichkeit besteht darin, in Cosmograph die Dateien "edges.csv" als data-file und "nodes.csv" als metadata file hochladen. (Tipp: "node label by" auf "label" umstellen.)
